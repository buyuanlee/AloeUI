## 开始使用
### 安装
#### npm安装
```npm
npm i aloe-ui -S 
```
### 快速上手
#### 引入AloeUI
##### 完整引入
在main.js中写入以下内容
```javascript
import Vue from 'vue';
import AloeUI from 'aloe-ui';
import 'element-ui/dist/index.css';
import App from './App.vue';

Vue.use(AloeUI);

new Vue({
  el: '#app',
  render: h => h(App)
});
```
