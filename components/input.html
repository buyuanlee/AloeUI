# Input-输入框

提供了三种基本状态的Input样式，默认状态、不可用状态以及警告状态 实例展示

示例代码

<aloe-input value="默认输入框"></aloe-input>
<aloe-input value="不可用输入框" disabled></aloe-input>
<aloe-input warning="请检查输入格式"></aloe-input>
